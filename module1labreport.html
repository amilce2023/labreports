Results & Analysis

This experiment is conducted by generating a Google Colab code that creates a square by moving a turtle on a canvas. The first part of the code consisted in importing the Turtle Library. This library allows the creation of the square by moving the turtle to create lines. The function initializeTurtle() sets up the turtle environment while the showturtle() makes the turtle visible to the user. The functions color() and bgcolor() were used to change the pen color and set the background color, respectively. Having initializing the turtle, the functions jump() and face() were used to move the turtle to a suitable position as desired. It is important to note that since we want to create a square, a "for" loop was used to to create the sides of the square. In this loop, the for_in range(4) allows to draw the 4 the four sides of the square. In this experiment each side was set to be 100 units long using the forward() function followed by the right() fucntion to turn the turtle a certain degrees to the right, and thus forming the corners of the square. Finally, the show() function displays the final drawing after the square was dawned.

In the modified portion of the code, the same functions and commands were used. However, the turtle starting position was set at the coordinates (500, 500) using the the command jump (500, 500).
